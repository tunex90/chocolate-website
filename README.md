# 🍫 The World of Chocolate

A simple, responsive static website that provides information about chocolate — its history, types, and fun facts. Built with plain HTML, CSS, and JavaScript, with no frameworks or dependencies.

---

## Live Preview

Open `index.html` directly in any modern web browser to view the site.

---

## Project Structure

```
chocolate-website/
├── index.html        # Main HTML page
├── style.css         # All styling and responsive layout
├── script.js         # Dynamically renders chocolate type cards
├── Chocolate 1.jpg   # Image used in the intro section
├── Chocolate 2.jpg   # Image used in the fun facts section
└── README.md
```

---

## Features

- Intro section covering what chocolate is and its origins
- Four chocolate type cards (Dark, Milk, White, Ruby) rendered dynamically via JavaScript
- Fun facts section with interesting trivia about chocolate
- Warm chocolate-themed colour palette
- Fully responsive layout — works on desktop and mobile
- No external libraries or dependencies

## Getting Started

No build tools or installations required.

1. Clone or download the repository:
   ```bash
   git clone https://github.com/<your-username>/chocolate-website.git
   ```
2. Navigate into the project folder:
   ```bash
   cd chocolate-website
   ```
3. Open `index.html` in your browser:
   ```bash
   # macOS
   open index.html

   # Windows
   start index.html

   # Linux
   xdg-open index.html
   ```

---

## Customisation

- To add or remove chocolate types, edit the `types` array in `script.js`
- To change colours, update the CSS variables/values in `style.css`
- To swap images, replace `Chocolate 1.jpg` and `Chocolate 2.jpg` with your own and update the `src` attributes in `index.html`

## Browser Support

Works in all modern browsers — Chrome, Firefox, Safari, and Edge.
